# React + Vite + TypeScript + Shadcn UI + Ahooks 云音乐 Web App

### 技术栈：

- React
- Vite
- TypeScript
- Shadcn UI
- Ahooks
- Taiwind CSS
- Zustand
- React Router
- Plyr
- Netease Cloud Music API

### 技术点

- 音频播放器 使用 Zustand 封装
- 视频播放器 使用 Plyr 封装组件
- 暗黑模式
- 滚动加载 使用 Ahooks useInfiniteScroll 封装
- 骨架屏
- 播放信息本地化存储
