# React + Vite + TypeScript + Shadcn UI + Ahooks 云音乐 Web App

### 技术栈：

- React
- Vite
- TypeScript
- Shadcn UI
- Ahooks
- Taiwind CSS
- Zustand
- React Router
- Plyr
- Netease Cloud Music API

- 音频播放器 使用 Zustand 封装
